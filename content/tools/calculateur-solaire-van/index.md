# Combien de watts installer sur le toit de votre van ?

Mode profils rapides pour une estimation en 30 secondes. Mode équipements réels pour dimensionner votre installation au watt près. Les vraies valeurs terrain : PVGIS toit plat corrigé -10%, rendement MPPT Victron 95% mesuré (pas les 98-99% du constructeur).

{{< calc-solaire-van >}}

## Mode rapide ou mode équipements réels ?

**Profils rapides** : choisissez un profil type (minimaliste, standard, confort, télétravail). Résultat immédiat. Idéal avant l'aménagement pour budgéter l'installation.

**Mes équipements réels** : renseignez chaque appareil (watts × heures d'utilisation par jour). Le calculateur totalise votre consommation journalière exacte. À utiliser avant d'acheter vos panneaux et votre batterie.

## Ce que le calculateur calcule vraiment

**Productible toit plat** : un panneau à plat sur un toit de van produit 8 à 12% de moins qu'un panneau incliné à 35° plein Sud. Ce calculateur applique la correction PVGIS-SARAH3 -10% — pas les chiffres optimistes inclinaison parfaite.

**Rendement MPPT terrain** : 95%, pas 98-99%. Les datasheets constructeurs mesurent en conditions de laboratoire. Le terrain, c'est des câbles pas toujours courts, des températures extrêmes, de la poussière de route.

**3 configurations** : Minimum, Recommandé, Confort. Choisissez selon votre budget et votre style de vie.
