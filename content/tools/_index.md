# Boîte à outils Sophie

Deux calculateurs pour préparer votre vanlife les yeux ouverts. Données terrain sourcées, formules visibles, méthodologies détaillées. Zéro fioriture.

## Vanlife & Budget

### [Calculateur budget vanlife mensuel](/tools/calculateur-budget-vanlife/)
Combien coûte vraiment la vanlife par mois ? 18 postes réels — carburant, stationnement, entretien, santé, connectivité. Chaque curseur est configurable, chaque chiffre est sourcé.
*Sources : Vanlife Factory 2025, camping-kerolland.fr 2026, prix-carburants.gouv.fr.*

### [Calculateur puissance solaire van](/tools/calculateur-solaire-van/)
Combien de watts installer sur le toit de votre van ? Mode profils rapides ou mode équipements réels pour dimensionner panneaux + batterie LFP avec les vraies valeurs terrain.
*Sources : PVGIS-SARAH3 JRC, Victron SmartSolar, VanPowerCalc, CARAPACE 2026.*
